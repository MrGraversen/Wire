# Wire

## Install

### Add JitPack Repository

```
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

### Add Maven Dependency

```
	<dependency>
	    <groupId>com.github.MrGraversen</groupId>
	    <artifactId>Wire</artifactId>
	    <version>0.0.1</version>
	</dependency>
```
